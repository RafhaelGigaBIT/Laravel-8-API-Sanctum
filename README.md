
## Sanctum Learn

API simples para cadastrar, mostrar e deletar produtos. 
Resistro, login e logout usando Sanctum.

## Instalação

git clone https://bitbucket.org/RafhaelGigaLucas/sanctum-learn-api.git

```bash
$ cd sanctum-learn-api
```
```bash
$ composer install
```
```bash
$ php artisan migrate
```
```bash
$ php artisan serve
```
